Això és una calculadora de terminal en català per MAC. Fet en C++
Com executar el programa: 
	a. Via terminal
		1. Obrir el terminal i posar: gpp -o <nom del arxiu creat (pots posar qualsevol nom)> <ruta de l'arxiu c++> && <ruta de l'arxiu creat anteriorment>
	b. Via drecera
		1. Buscar el arxiu que es diu: CalculadoraDrecera
		2. Fer-li doble click
